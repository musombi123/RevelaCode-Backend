# 🔐 Security Policy

## 📌 Supported Versions

| Version | Supported          |
|--------|--------------------|
| 5.1.x  | ✅ Supported        |
| 4.0.x  | ✅ Supported        |
| 5.0.x  | ❌ Not supported    |
| <4.0   | ❌ Not supported    |

We recommend always using the latest supported version for maximum security.

---

## 🛡️ Reporting a Vulnerability

If you discover a security issue in this project:

- Please **DO NOT** create a public GitHub issue.
- Instead, email us directly at:  
  ✉️ **musombiwilliam769@gmail.com** (or use another security contact email)
- Include as much detail as possible: version, steps to reproduce, impact, etc.

We aim to respond **within 72 hours** and will keep you updated about:
- Confirmation of the issue
- Mitigation steps
- Planned release timeline (if applicable)

---

## ✅ Responsible Disclosure

We deeply appreciate responsible disclosure!  
If the vulnerability is valid, you’ll be credited in the release notes unless you’d prefer to stay anonymous.

Thank you for helping keep this project and its users safe! ✨
